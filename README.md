# Sistema de Gestão Académica

Bem-vindo(a) ao **Sistema de Gestão Académica** – uma plataforma moderna, intuitiva e totalmente responsiva para gestão completa de instituições de ensino.

Este repositório contém o **frontend** da aplicação, desenvolvido com as tecnologias mais atuais do ecossistema React/Next.js.

## Funcionalidades principais

- Gestão completa de alunos, turmas, professores e funcionários
- Controle de matrículas, frequências e notas
- Calendário académico e horário de aulas
- Emissão de boletins e relatórios
- Painel administrativo com permissões por perfil (Admin, Secretário, Professor, Aluno/Encarregado)
- Design 100% responsivo (funciona perfeitamente em telemóvel, tablet e desktop)
- Tema claro e escuro automático

## Tecnologias utilizadas

- **React** (App Router + Server Components)
- **TypeScript**
- **Tailwind CSS** + **Shadcn/UI** (componentes bonitos e acessíveis)
- **Lucide React** (ícones modernos)
- **Zod** (validação de formulários)
- **React Hook Form**
- **TanStack Query** (gestão de estado do servidor)
